Einführung

Die Registermaschine ist ein einfaches Modell eines Computers, das aus einer begrenzten Anzahl von Registern und einem Programm besteht, das in einer einfachen Assemblersprache geschrieben ist. Diese Maschine kann grundlegende arithmetische Operationen und Kontrollflussanweisungen ausführen.

Um Kommentare in Ihrem Programm hinzuzufügen, verwenden Sie das Semikolon-Zeichen oder Raute-Zeichen (;, #). Alles, was nach dem Kommentarzeichen in einer Zeile steht, wird als Kommentar betrachtet und von der Registermaschine ignoriert.

Programmfluss

Die Registermaschine unterstützt bedingte und unbedingte Sprungbefehle, die es ermöglichen, den Programmfluss zu steuern. Bedingte Sprünge überprüfen den Wert eines Registers und springen zu einer bestimmten Programmzeile, wenn die Bedingung erfüllt ist.

  • JMP ##
    Springt zur angegebenen Programmzeile (## ist die Speicherzelle von 0 - SPEICHERLAENGE)
  • JEZ ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator 0 ist.
  • JNE ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator ungleich 0 ist.
  • JLZ ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator kleiner 0 ist.
  • JLE ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator kleiner oder gleich 0 ist.
  • JGZ ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator größer 0 ist.
  • JGE ##Springt zur angegebenen Programmzeile, wenn der aktuelle Wert im Akkumulator größer oder gleich 0 ist.
  • HLT 99 Beendet das aktuelle Programm.
 Lade- und Speicherbefehle Verarbeitungsbefehle
  • ADD ##
    Addiert auf den aktuellen Wert im Akkumulator den Wert in der angegebenen Speicherzelle (##). Das Ergebnis landet wieder im Akkumulator. ADD 00 addiert den Wert im Akkumulator mit sich selbst.
  • SUB ##
    Subtrahiert den Wert in der angegebenen Speicherzelle (##) vom aktuellen Wert im Akkumulator. Das Ergebnis landet wieder im Akkumulator. ADD 00 addiert den Wert im Akkumulator mit sich selbst.
  • MUL ##
    Multiplizert den Wert im Akkumulator mit der angegebenen Speicherzelle (##). Das Ergebnis landet wieder im Akkumulator. ADD 00 addiert den Wert im Akkumulator mit sich selbst.
  • DIV ##
    Dividiert den aktuellen Wert im Akkumulator mit dem Wert in der angegebenen Speicherzelle (##). Das Ergebnis landet wieder im Akkumulator. ADD 00 addiert den Wert im Akkumulator mit sich selbst.
Transportbefehle
  • LDA ##
    Lädt den Wert der angegebenen Speicherzelle (##) in den Akkumulator.
  • LDK ##
    Lädt die angegebenen KONSTANTE (##) in den Akkumulator.
  • STA ##
    Speichert den Wert im Akku in die angegebenen Speicherzelle (##).
  • INP ##
    Speichert den Wert vom INPUT in die angegebenen Speicherzelle (##). INP 00 speichert den Wert vom INPUT direkt in den Akkumulator.
  • OUT ##
    Schreibt den Wert der Speicherzelle (##) auf den OUTPUT. OUT 00 schreibt den aktuellen Akkumulatorwert auf den OUTPUT.
